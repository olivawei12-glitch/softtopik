TOPIK I Core Grammar

Start with essential beginner grammar structures.

-이에요 / -예요

Used to say “to be” in present tense.

학생이에요. (I am a student.)

-은 / -는

Topic marking particles.

저는 학생이에요.

-이 / -가

Subject marking particles.

비가 와요.

-을 / -를

Object marking particles.

책을 읽어요.
